# Steps To Deploy
1. Clone the repository: git clone https://github.com/Jaggi037/odyssey_plan.git
2. Start the frontend: cd frontend -> npm i -> npm run dev
4. Start the apiServer in parallel terminal: cd backend -> cd apiServer -> npm i -> nodemon app.js
5. Start the realTimeServer in parallel terminal: cd backend -> cd realTimeServer -> npm i -> nodemon app.js
6. Start the MediaServer in parallel terminal: cd backend -> cd mediaServer -> npm i -> nodemon app.js
